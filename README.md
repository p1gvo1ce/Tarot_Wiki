# Tarot Card Application

**О проекте:**
Это приложение для работы с колодами Таро, написанное на Python с использованием [Kivy](https://kivy.org/). Проект позволяет создавать и редактировать колоды, а также просматривать карты вместе с их описаниями и изображениями.

---

## Фичи проекта
- **Редактирование колод и карт**: добавляйте новые карты, изменяйте описания и прикрепляйте изображения.
- **Просмотр карт**: выбирайте колоду и карту для быстрого просмотра описания и картинки.
- **Интеграция с буфером обмена**: возможность вставлять изображения напрямую из буфера обмена.
- **Автоматическая инициализация базы данных** при запуске.

---

## Требования

- Python 3.7+
- Windows (для корректной работы выбора дисков в интерфейсе)

---

## Установка и запуск

### Первоначальная настройка
1. **Клонируйте репозиторий** или скачайте исходный код в удобное для вас место.
2. Найдите файл `setup_and_run.bat` в корне проекта.
3. Дважды щёлкните по `setup_and_run.bat` или запустите его из командной строки.  
   Скрипт:
   - Создаст виртуальное окружение.
   - Установит все необходимые зависимости.
   - Запустит `run_app.bat` для старта приложения.

### Последующие запуски
Для обычного запуска приложения используйте `run_app.bat`:
1. Дважды щёлкните по `run_app.bat` или запустите его из командной строки.  
   Скрипт:
   - Проверит наличие виртуального окружения.
   - Если окружение существует, активирует его и запустит приложение.
   - Если окружение отсутствует или произошла ошибка, запустит настройку через `setup_and_run.bat`.

## Как пользоваться

- **Редактор карт:**
  1. Перейдите на экран редактора.
  2. Введите название колоды и выберите карту.
  3. Добавьте описание и выберите изображение (или вставьте из буфера обмена).
  4. Нажмите "Сохранить" — и карта сохранится в базе данных.

- **Просмотр карт:**
  1. На экране просмотра выберите колоду.
  2. Затем выберите карту из предложенных спиннеров.
  3. Описание и изображение карты отобразятся автоматически.

## Возможные проблемы и решения

- **Не запускается `run_app.bat`?**
  - Убедитесь, что Python установлен и добавлен в PATH.
  - Скрипт проверяет наличие виртуального окружения и при необходимости запускает настройку.

- **Ошибки при установке зависимостей:**
  - Проверьте корректность файла `requirements.txt`.
  - Убедитесь, что интернет-соединение активно для скачивания пакетов.
